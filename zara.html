Zara UI redesign

Case study on the user interface, adding dark mode and auto-suggest search bar feature

Timeline

Sept 2024 - Dec 2024 (Work in progress)

Role

Solo student Project

Overview

This redesign of Zara’s website aims to improve user experience by introducing dark mode and an auto-suggest feature in the search bar.

Problem Statement

The current Zara website lacks a dark mode option, which can improve readability in low-light settings, and doesn’t support auto-suggest in the search bar, making it harder for users to quickly find products.

Design and Prototyping

Using Figma, I created a wireframe incorporating a dark mode with contrasting colors for readability and the elements needed for the design. These included the toggle switch, buttons, and others.​​​​​​​

Usability Testing

I tested the prototype with a small group from the target audience, gathering feedback on the new dark mode and search functionality. Users responded positively to the added convenience and visually comfortable dark mode

Result and Outcome

Overall, the project enhanced accessibility and provided a more user-friendly experience on the Zara website.
